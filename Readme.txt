1. Check out git project
2. Import project in IDE (Eclipse)
3. Run Maven build.
4. Run MAven install.
5. Run com.up42.features.UP42APIApplication as java application.
6. Tomcat started on port(s): 8084 (http) with context path '/feature-api'
7. Open chrome browser
8. NAvigate to URL - http://localhost:8084/feature-api/swagger-ui.html
